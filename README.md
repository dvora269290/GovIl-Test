**GovIl-Test**

Automated end-to-end testing configuration and sample tests using Cypress for the GovIl project.

This repository contains test configuration and example specs to run UI/UX tests against a target web application using Cypress.

**Project Overview**

GovIl-Test is a Cypress test suite designed to:

Run end-to-end tests on key user flows in a web application

Automate UI validation and regression checks

Provide a baseline for expanding test coverage


 **Features**

Cypress configuration for test automation

Example test specs (add more as needed)

Cross-platform test execution

Easy to extend with new test cases

**Getting Started**
**Prerequisites**

Install Node.js:

node -v


Install dependencies:

npm install

**Running Tests**

Launch Cypress UI:

npm install cypress --save-dev

npx cypress open
